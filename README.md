# TechMaster

---
## Chạy prj 01
* Yêu cầu với môi trường trước khi chạy
  - Node.js và npm 
  - PostgresSQL cho database
  - Golang
  - Air (live reload cho golang backend)
* Các bước chạy
  1. Clone project này về, sau đó:
     ```bash
     cd TechMaster/01
     go build -o ./application main.go
     go run main.go 
     ```
     Backend sẽ được chạy ở port 8081
